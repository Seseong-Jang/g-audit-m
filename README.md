# g-audit-m

support version: Mysql 5.5 ↑, MariaDB 10.0.21 ↑

+ extract tar.gz file

`
    tar -zxvf g-audit-logging.tar.gz
`

+ install g-audit-m

`
    ./install_audit.sh {ROOT_PW}
`

+ uninstall g-audit-m

`
    ./uninstall_audit.sh {ROOT_PW}
`

+ for scheduling script - filtering failed connect log

`
    ./filter_audit_log.sh
`
